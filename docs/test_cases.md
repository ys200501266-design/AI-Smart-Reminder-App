# 测试用例

| 测试场景 | 输入 | 预期结果 | 实际结果 | 优先级 |
| --- | --- | --- | --- | --- |
| 手动创建提醒 | 标题：交作业；日期时间：未来时间 | 创建成功，列表出现新提醒 | 待测试 | P0 |
| 日期格式错误 | 2026/06/01 | 提示日期或时间格式不正确 | 待测试 | P0 |
| 过去时间 | 今天已经过去的时间 | 不重复提醒创建失败 | 待测试 | P0 |
| 每天重复提醒 | 每天 15:00 喝水 | 创建每天重复通知 | 待测试 | P0 |
| 每周重复提醒 | 每周当前星期 09:00 开会 | 创建每周重复通知 | 待测试 | P0 |
| 10 秒测试提醒 | 点击“10 秒测试提醒” | 10 秒后出现系统通知 | 待测试 | P0 |
| 删除提醒 | 删除一条待提醒 | 列表移除，通知取消 | 待测试 | P0 |
| 标记完成 | 点击“标记完成” | 状态变为已完成，通知取消 | 待测试 | P1 |
| 解析明天提醒 | 明天早上九点提醒我交作业 | 日期为明天，时间为 09:00，事项为交作业 | 待测试 | P1 |
| 解析今晚提醒 | 今晚八点提醒我背单词 | 时间为 20:00，事项为背单词 | 待测试 | P1 |
| 解析每天提醒 | 每天下午三点提醒我喝水 | repeat_type 为 daily，时间为 15:00 | 待测试 | P1 |
| 解析失败兜底 | 随便提醒一下 | 提示手动确认或修改 | 待测试 | P1 |
| 拒绝通知权限 | 首次弹窗选择拒绝 | 创建失败并提示开启权限 | 待测试 | P0 |
