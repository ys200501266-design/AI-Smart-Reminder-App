# 用户故事

| 用户类型 | 用户故事 | 价值 |
| --- | --- | --- |
| 经常忘事的学生 | 作为一个经常忘事的学生，我希望输入一句话就能创建提醒，以便我不用手动设置复杂闹钟。 | 降低创建成本 |
| 准备考试的学生 | 作为一个正在复习考试的学生，我希望每天固定时间提醒我背单词，以便形成学习习惯。 | 支持重复提醒 |
| 实习生 | 作为一个实习生，我希望提醒内容能显示具体事项，以便我看到通知时知道要做什么。 | 提升通知可用性 |
| Demo 录制者 | 作为一个作品集展示者，我希望有 10 秒测试提醒，以便录屏时快速展示通知能力。 | 提升展示效率 |
| 普通用户 | 作为一个普通用户，我希望能删除不需要的提醒，以便列表保持清晰。 | 管理提醒 |
| 普通用户 | 作为一个普通用户，我希望完成任务后能标记完成，以便知道哪些事情已经处理。 | 任务闭环 |
| AI 产品经理面试者 | 作为一个 AI 产品经理求职者，我希望项目能体现 AI 理解层和执行层设计，以便在面试中讲清楚产品逻辑。 | 展示 AI PM 能力 |
